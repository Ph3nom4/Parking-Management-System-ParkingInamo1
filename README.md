# Parking-Management-System-ParkingInamo1
Simple Parking management system features a entry vehicle- exit vehicle and admin management setup with incrementing price depending on hours 

Features:
- Vehicle Entry using Dgv
- Exit Vehicle and compute total time stayed with succeeding rates of first 2 hours and additional per hour
- Parking Lot Visual form shows real time slot available and occupied space
- Manage System Form allows removal and editing of Vehicle parked in Main Dgv
